Objetivos del Proyecto

principio:Preparion de los datos 

- Implementar procesos ETL para limpieza de datos
- Diseñar modelo dimensional (Data Warehouse)
- Aplicar técnicas de limpieza y transformación de datos
- Garantizar calidad y consistencia de los datos

Unidad 3: Análisis Supervisado  
- Desarrollar modelos predictivos de regresión
- Implementar y comparar algoritmos de machine learning
- Evaluar métricas de rendimiento de modelos
- Optimizar hiperparámetros para mejor desempeño

Tecnologías Utilizadas

- Lenguaje: Python 3.8+
- Librerías principales: 
  - `pandas` - Manipulación de datos
  - `scikit-learn` - Machine Learning
  - `matplotlib` y `seaborn` - Visualización
  - `numpy` - Cálculos numéricos
- Control de versiones: Git + GitHub
- Metodología: CRISP-DM para ciencia de datos

Caso de Estudio: Tienda Departamental "MegaStore"

Contexto del Negocio
Análisis de datos transaccionales de una tienda departamental para optimizar:
- Pronóstico de ventas e ingresos
- Gestión de inventario
- Estrategias de pricing
- Planificación comercial

Datasets
- Transaccionales: Ventas, productos, clientes, sucursales
- Temporales: Fechas de venta, patrones estacionales
- Geográficos: Ciudades de clientes y sucursales
